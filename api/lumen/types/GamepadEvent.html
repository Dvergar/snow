GamepadEvent

Information about a gamepad event\n\nraw : The raw event from the system, unmodified \ntimestamp : The time in seconds when this touch event occurred, use for deltas \ntype : The button id, if the event type is button \nstate : The state this event is in \nwhich : The id of the gamepad this event comes from \nbutton : The button id, if the event type is button \naxis : The axis id, if the event type is button \nvalue : The axis value, if the event type is axis
