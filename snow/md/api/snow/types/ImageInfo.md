<h1>ImageInfo</h1>
<small>`snow.types.ImageInfo`</small>

Information about an image file/data

`id` : source asset id
`bpp` : used bits per pixel
`width` : image width
`height` : image height
`bpp_source` : source bits per pixel
`data` : image data
